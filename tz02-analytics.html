Завдання 02 · Аналітика

З публічних джерел маю SimilarWeb — 81K відвідувачів за березень 2026, 79% платний пошук, bounce rate 58.7%. Глибші дані (CTR на партнера, внутрішні конверсії) публічно недоступні через redirect без referrer. Тому для unit economics використовував датасет з третього завдання виключно як джерело залежностей — CR і RPC, але не як абсолютні обсяги.

Методологія: зафіксував три сценарії CTR кнопки "Visit Site" (8% / 15% / 20%), через bounce rate зворотно порахував implied visits і paid spend, і порівняв з реальним revenue з датасету. Break-even CPC у найкращому сценарії — $0.54, тоді як ринок стоїть $1.20–2.50. Висновок: без покращення CR на сайті масштабувати платну закупівлю збитково при будь-якому сценарії.

Висновок

У всіх трьох сценаріях збитково при ринковому CPC $1.20–2.50. Break-even CPC максимально $0.54 — тобто потрібен трафік в 2-5x дешевший за ринок, або потрібно кратно покращити CR на сайті. CAC при $1.80 і CTR 20% = $2,496 при CPA виплаті $743 — збиток $1,753 на кожен продаж. Саме тому продуктові механіки (завдання 02) — не nice-to-have, а умова виживання при платній закупівлі.

Припущення та джерела
SimilarWeb 81,136 visits/міс, 79.3% Paid Search — реальні дані березень 2026. Bounce rate 58.7%, Visit duration 1:12.
Датасет CR click→lead 5.8%, CR lead→sale 11.8%, RPC $5.14 — реальні конверсії з тестового датасету Asian Dating (70,776 кліків, 490 продажів).
Бенчмарк CPC $1.20–2.50 — галузевий бенчмарк Google Ads для dating/Asian keywords (США). Джерело: WordStream Dating Industry, 2024.
Припущення Partner CTR 25% — з DOM аудиту сайту: 3 CTA на картку (VISIT SITE, READ REVIEW, BROWSE SINGLES), середній CTR для affiliate review сайтів 20-30%.
Ліміт Outgoing traffic недоступний — SimilarWeb показав "No results" для outgoing links через redirect без referrer (/go/ endpoint). Реальний CTR на партнера — тільки з внутрішньої аналітики.
